Device and System Data types

The application may collect basic, non-personal technical information such as device model, operating system version, and anonymised usage statistics or error logs. This information is used solely to monitor performance, improve reliability, and ensure compatibility across different mobile devices and environments.

Device Data Collection & Storage

When interacting with the application, user-entered data is processed temporarily within secure in-memory structures. These values remain available only while the application session is active. Once the app is closed or refreshed, the information is cleared unless the user deliberately enables persistent storage features. No personal or sensitive data is uploaded to external servers, and the app only communicates with remote endpoints when necessary (e.g., retrieving sample data). Location data and unrelated identifying information are never collected or transmitted.
